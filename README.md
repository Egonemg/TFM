# TFM
Este repositorio recoge el desarrollo de mi Trabajo Final de Máster, centrado en el reconocimiento automático de emociones en la voz mediante técnicas de Machine Learning y Deep Learning.
El objetivo es desarrollar y evaluar modelos capaces de identificar estados emocionales a partir de señales de voz, con posible aplicación en contextos de salud y bienestar.

Presenta el:
- Preprocesado de audio
- Extracción de características
- Entrenamiento y evaluación de modelos
- Análisis de resultados

Tecnologías utilizadas:
- Pyhton
- Google Colab
- Machine Learning
- Deep Learning
- Procesamiento de audio y voz

Este es un proyecto de carácter académico. Algunos datos y detalles han sido anonimizados.
El enfoque técnico y las decisiones de diseño pueden ampliarse y explicarse en detalle durante una entrevista.
